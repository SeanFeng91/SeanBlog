# 现有的思考
1. 市面上AI的课程都还是面向成人，面向青少年的课程很少。
2. 未来是AI的时代，它就跟互联网一样会渗透到生活的方方面面。改变知识的获取方式，改变学习的方式。
3. AI一定会是一个高效的工具，能够使用和不去使用的差别会越来越大。能够更好的使用的一定是受益者。
4. 

# 现有的挑战和困难
1. 怎么了解到真正的需求，或者怎么去验证自己的想法；
- 与老师联系
- 与有孩子的家长聊天
- 与青少年进行具体体验，搭建体验的课程
2. 怎么去构建一个完整的课程体系，包括内容、形式、方法、工具等；
3. 怎么去构建一个完整的教学体系，包括老师、学生、家长、学校等；
4. 怎么去构建一个完整的商业模式，包括收费、盈利、市场等；
