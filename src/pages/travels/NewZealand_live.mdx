# 新西兰天气

新西兰南岛位于南纬40°至47°之间，属于温带海洋性气候，气候受海洋、大气环流、南极洋流的影响较大，天气变化较频繁。10月份是春季中期，此时南岛的气候逐渐变暖，以下是南岛10月份的主要气候特征：
下图中就可以看出，南纬度靠近南极的云量还是非常的丰富的。好天气只能是在两个云带之间的空隙。
<image src="/SeanBlog/pictures/20240926-NZTripInfo_blog/NZ69.png" alt="NZ"></image>

1. 大气环流和季风
新西兰位于中纬度西风带，受西风带控制，大气环流的主导作用显著。西风带从南太平洋的广阔水域带来潮湿空气，在新西兰的南阿尔卑斯山遇到地形抬升，造成大量降水，尤其是西海岸地区。此外，偶尔会有来自南极的冷空气带来短暂的降温和降雪。
新西兰南岛不受典型的季风影响，主要受制于周围的海洋影响，但在10月份，风向主要为西风，较为稳定。

2. 降雨
南岛西海岸地区降雨丰沛，尤其是南阿尔卑斯山脉的西侧，年均降雨量可以达到6000毫米。10月份，由于南岛正处于春季，西部地区降雨较多，月平均降水量约在200至400毫米左右。而东海岸如基督城（Christchurch）等地，降水量较少，月降雨量约为50至100毫米。

3. 气温
10月份南岛气温逐渐回升，平均气温在10°C至15°C之间，北部的尼尔森（Nelson）地区温度较高，日间气温可达18°C，夜间气温通常在5°C至10°C。南部地区如皇后镇（Queenstown）及菲奥尔德地区气温较低，日间气温在12°C左右，夜间可能降至0°C至5°C之间。
