# 记账软件切换
离开口袋记账最主要的原因是它开始无休止的广告模式，而且包年的费用是88元，远比一木的终身会员要高。
而且所谓的会员也只是多了免广告、数据同步、头像、皮肤、标识等无关紧要的，说白了就是你离不开我口袋记账，
广告烦死你，看你买不买我的会员。好在非会员能导出账单，系统花了几天时间才把导出的账单发给我的邮箱。
<image src="/SeanBlog/pictures/20240916-Yimu_blog/Yimu03.png" alt="Yimu App"></image>

因为模板格式的差异，一木把消费类型分成了两级，比如购物消费-淘宝，而口袋记账是直接一个类型，约等于二级类别，所以
需要大致将几千条消费类型加上一级类型，不然直接导入就会变成：口袋记账-淘宝，或者其他-淘宝。
整个导入估计花费了大概三四个小时的时间。终于把口袋记账的内容完全导入了新下载的一木记账。
<image src="/SeanBlog/pictures/20240916-Yimu_blog/Yimu01.png" alt="Yimu App"></image>

本身软件有很多令人惊喜的功能，因为是国内的开发团队做的，考虑了比如支持支付宝、微信等截图记账功能。
不论是不是会员都承诺了永远没广告。还支持面部解锁。这是个功能我觉得像石墨文档、记账软件这些带有一些隐私性记录
的app所需要的功能。最终决定花费了98元购买终身会员。简单的体验了一个好用的功能是智能记账，
可以用一段文字描述，它就可以区分出不同的信息进行分类记录，这样就简化了记录的环节。摘录一段对它功能的概括：

> 一木记账丰富的记账方式，语音记账、一句话文字记账、自动记账、模板记账、周期记账、桌面快捷记账、手动记账，只为记账而生。强大的资产管理，资产账户管理、信贷账户分期管理、还款提醒、流水悬浮窗对账、报销、债务、基金股票收益自动同步。高效的日历查阅，每日收支、还款提醒，尽在日历视图一目了然！一木记账独有的导入导出，支持第三方平台账单截图一键导入，自动识别账单，支持多个第三方平台Excel账单导入，支持高度自由的账单导出。举重若轻的功能，协同记账、预算设置、标签、WebDAV备份、云端备份、退款、自定义主题、自定义图标、快捷手势、丰富的小部件，自由的个性化设置。不带任何理财信息广告，只为记账而生！

不过统计分析的功能，只能说各有所长吧。阅读趋势部分口袋记账绘制了折线图，而一木用的是柱状图。口袋记账有每个类别
近三个月的趋势对比。也有成员的概念。一木则把日历模式和瀑布模式与统计分析都做成了平行级别放在下方组件中。可能是
因为记账类型更全面，除了收入支出等常规指标，还有转账、还款、收款、应收、应付、手续费、报销、退款、优惠等等维度
统计。资产走势、标签占比等等。应该是对非常深度用户的一些账单统计功能补充。

关于一木记账，还有一些关于资产、还贷等功能还不能够重复的理解记录的方式还没有特别的搞懂，
包括关联记账的意义，自动记账的体验、标签的意义等等。似乎缺一个用户手册或者说明。
开发的人一定是想着这些功能只要去用就可以理解是干啥的。【才发现有个关于软件的Gitbook，
到时候看一下再看看还有哪些新的功能】

因为我也是近一两年才开始相对持续且精细的记账。想要补齐之前的数据就要去各个银行找历史账单。
说到这个账单，一个跟个人信息这么相关的数据，有些银行在软件端居然也只能查3年内的信息，
再往前的都不知道存在哪里，可能的确数据量大。招商银行在联系客服之后花费了近两周多的时间，
才把三年前的账单数据导出发送过来。据说最多可以往前十五年。可能这也就是大多数年轻人使用银行卡的年限了。
但是要是再过一些年份，前面的数据如果没有导出可能就再也找不到了。旅行软件的购买机票酒店记录，
淘宝的记录，大致也是如此。

# PDF账单转Excel

在补充记录过往账单的过程中也遇到了一个强烈的需求，就是把银行的pdf账单转换成excel文件。
目前在用的笔记软件石墨也开启它的收费之旅，pdf转excel也在会员权限之内。
好在ilovepdf一直不离不弃的不收费，具备了所有跟pdf相关的功能。
<image src="/SeanBlog/pictures/20240916-Yimu_blog/Yimu04.png" alt="Yimu App"></image>
不过转化的excel还是需要人为简单的处理，会有很多空的列、行和无用的账单信息。

跟piksou聊创业项目的过程也发现了有类似产品。这样的情况大致是试用一个月一个月导入的情况。
但是真的数量多了，你要对转换之后的表格内每一笔都去标注消费类型，也是个不小的工作。
<image src="/SeanBlog/pictures/20240916-Yimu_blog/Yimu02.png" alt="Yimu App"></image>
