# Tripo构建模型
这一轮更新之后，[Tripo](https://www.tripo3d.ai/app)给每个新注册的用户提供了600积分。如果是文字描述生成模型花费60积分，如果是图片生成模型花费20积分。每次生成模型如果不满意可以免费再生成，不会额外扣几分。
但一旦生成结束进入预览，好像就不可以再重新生成模型了。
