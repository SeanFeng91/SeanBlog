# MDX 文档写作规范

MDX 是一种强大的文档格式，它结合了 Markdown 的简洁性和 JSX 的灵活性。本文将介绍我们的 MDX 文档写作规范，并展示一些常用的功能和样式，包括一些高级的 CSS 和 JSX 功能。

## 基本格式

### 标题

使用 `#` 符号来创建标题，从一级标题 `#` 到六级标题 `######`。例如：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

### 段落和换行

段落之间使用空行分隔。如果需要在段落内换行，可以在行末添加两个空格。

这是第一个段落。

这是第二个段落。

### 强调

使用 `*` 或 `_` 来表示斜体，使用 `**` 或 `__` 来表示粗体。例如：

*斜体文本* 或 _斜体文本_

**粗体文本** 或 __粗体文本__

***粗斜体文本*** 或 ___粗斜体文本___

## 高级样式



### 文字高亮

我们可以直接在MDX文件中定义一个简单的高亮组件：


<Highlight color="#25c2a0">这是绿色高亮文本</Highlight>

<Highlight color="#1877F2">这是蓝色高亮文本</Highlight>

### 动态效果

我们可以在MDX文件中直接定义一个简单的淡入效果组件：

<FadeIn>
  这段文字会淡入显示
</FadeIn>


### 交互式组件

我们可以在MDX文件中直接定义一个简单的计数器组件：
```
export const Counter = () => {
  const [count, setCount] = React.useState(0);
  return (
    <div>
      <p>你点击了 {count} 次</p>
      <button onClick={() => setCount(count + 1)}>
        点击我
      </button>
    </div>
  );
};
```
